MuK Scroll Up Button

Button to jump to the top of the page

MuK IT GmbH - www.mukit.at

Overview

Adds a button to the bottom of the page to jump back up to the top of the page. The button can be customized using the following attributes.

Attribute Default Description
scroll_distance 300 Distance from top/bottom before showing element (px)

Customize

Change the settings of the scroll-up button by editing the template file through the HTML/CSS editor.

Change the appearance of the scroll-up button by editing the less file through the HTML/CSS editor.

Settings

Turn the scroll-up button on or off by toggling the slider inside the customize settings dropdown. After the page has reloaded the button will be either invisible or visible according to your settings.
