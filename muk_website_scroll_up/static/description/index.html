MuK Scroll Up Button

Button to jump to the top of the page

MuK IT GmbH - www.mukit.at

Overview

Adds a button to the bottom of the page to jump back up to the top of the page. The button can be customized using the following attributes.

scroll_distance - default: 300 Distance from top/bottom before showing element (px)
scroll_from - default: "top" 'top' or 'bottom'
scroll_speed - default: 300 Speed back to top (ms)
easing_type - default: 'linear' Scroll to top easing
animation - default: 'fade' Fade, slide, none
animation_speed - default: 300 Animation speed (ms)
scroll_trigger - default: false Set a custom triggering element. object
scroll_target - default: false Set a custom target element for scrolling to.
scroll_text - default: <i class='fa fa-chevron-circle-up' /> Text for element
scroll_title - default: false Set a custom <a> title if required.
scroll_img - default: false Set true to use image
active_overlay - default: false Set CSS color to display scrollUp active point, e.g '#00FFFF'
z_index - default: 300 Z-Index for the overlay

Customize

Change the settings of the scroll-up button by editing the template file through the HTML/CSS editor.

Change the appearance of the scroll-up button by editing the less file through the HTML/CSS editor.

Settings

Turn the scroll-up button on or off by toggling the slider inside the customize settings dropdown. After the page has reloaded the button will be either invisible or visible according to your settings.
